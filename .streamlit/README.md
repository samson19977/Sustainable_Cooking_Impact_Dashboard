# Sustainable Cooking Impact Dashboard

## Overview
Interactive dashboard for monitoring DelAgua Stove Adoption Programme in Rwanda's Northern Province. This dashboard provides data-driven insights for targeted interventions to improve fuel reduction outcomes.

## Key Features
- **Real-time Monitoring**: Track 7,976 households across 5 districts
- **Predictive Analytics**: Identify high-risk households using machine learning
- **Geographic Insights**: Map-based visualization of adoption patterns
- **Actionable Recommendations**: Tiered intervention strategies
- **Environmental Impact**: Quantify fuel savings and COâ‚‚ reduction

## Insights from DelAgua Report
- 47.8% of households (3,809) below 30% adoption target
- Top risk predictors: Distance to market, district location, household size
- Potential additional savings: 652 tons of fuel annually with targeted interventions
- Geographic clustering identified 2 major low-adoption zones
